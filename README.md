## 🔑 Key Points
    spinner-border → rotating circle spinner.
    spinner-grow → pulsing/growing spinner.
    Color classes (text-primary, text-success, etc.) change spinner color.
    Size control → use spinner-border-sm or custom width/height.
    Accessibility → always include hidden text for screen readers.
    Buttons → spinners can be embedded inside buttons to show “loading” state.
